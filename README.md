# sud_zbir_rozrah

Програма створена для розрахунку судового збору при поданні деяких документів до Українських судів.

Простенький і кривенький GUI створено за допомогою бібліотеки tkinter. У разі необхідності в оновлені, нажаль прийдеться хард-кодити внесені зміни, так само, за потреби
внести зміну у константи(Рік з якого вираховується прожитковий мінімум, чи процентна ставка інстанцій) також необхідно хардкодити.

Написано на замовлення юридичної служби в/ч А3913 Збройних Сил України.
