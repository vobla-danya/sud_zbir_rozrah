# sud_zbir_rozrah

Програма створена для розрахунку судового збору при поданні деяких документів до Українських судів.

GUI на основі бібліотеки Customtkinter. Збірка у бінарний файл за допомогою cx_Freeze.
За необхідності внесення змін у константи, необхідно змінити config.toml.

Написано на замовлення юридичної служби в/ч А3913 Збройних Сил України.
